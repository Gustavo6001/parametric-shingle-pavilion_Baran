PROJECT TITLE:
Parametric Curved Shingle Pavilion

GOAL:
Create a parametric curved pavilion covered with repeated shingle panels. The pavilion should be controlled using sliders in Grasshopper and generated with Python.

INPUT PARAMETERS:
- pavilion_width
- pavilion_height
- pavilion_depth
- curvature_amount
- row_count
- column_count
- shingle_width
- shingle_height
- overlap_amount
- tilt_angle
- support_height
- base_width
- base_depth
- base_thickness

PSEUDO CODE:

START

1. Define global pavilion parameters:
   - width
   - height
   - depth
   - curvature amount

2. Define grid parameters:
   - number of rows
   - number of columns

3. Define shingle parameters:
   - shingle width
   - shingle height
   - overlap amount
   - tilt angle
   - support height

4. Define base parameters:
   - base width
   - base depth
   - base thickness

5. Create a curved vertical pavilion surface:
   - Use width as the horizontal direction
   - Use height as the vertical direction
   - Use depth and curvature amount to bend the surface
   - The surface leans backward and curves softly

6. Create a local frame system:
   - For each point on the surface
   - Calculate the local X direction
   - Calculate the local Z direction
   - Calculate the surface normal direction

7. Divide the surface into a grid:
   - Rows control the vertical subdivision
   - Columns control the horizontal subdivision
   - Each grid cell becomes a location for one shingle

8. Apply fish-scale row offset:
   - If the row index is even, keep the original position
   - If the row index is odd, shift the shingle by half the horizontal spacing
   - This creates a staggered scale pattern

9. Create shingle panels:
   - For each grid location
   - Create one rectangular or slightly tapered panel
   - Place the panel on the local surface frame
   - Move it outward from the surface using the normal direction

10. Apply shingle tilt:
   - Rotate or move the upper part of the shingle outward
   - Use the tilt angle parameter
   - This creates depth and shadow between shingles

11. Create support spacers:
   - For each shingle
   - Create a support line or spacer from the surface to the shingle
   - Use support height as the distance

12. Create vertical structural ribs:
   - Generate vertical curves along the pavilion surface
   - These ribs define the main structural frame

13. Create horizontal purlins:
   - Generate horizontal curves across the pavilion surface
   - These connect the ribs and support the shingles

14. Create side frames:
   - Add structural lines on the left and right edges
   - These stabilize the pavilion form

15. Create base platform:
   - Create a rectangular platform under the pavilion
   - Use base width, base depth, and base thickness
   - Align the pavilion with the platform

16. Output geometry:
   - Output shingles
   - Output supports
   - Output ribs
   - Output purlins
   - Output base

17. Output a text report:
   - Student name
   - Project title
   - Number of rows
   - Number of columns
   - Total shingles generated

END
