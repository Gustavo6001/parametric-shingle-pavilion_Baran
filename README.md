# Parametric Shingle Pavilion Generator

## Project Description

This project is a beginner-friendly parametric geometry generator for a pavilion covered with repeated shingle panels.

The project follows a top-down computational design workflow:

1. Define a design goal
2. Write pseudo-code
3. Create an AI development plan
4. Build the code with AI assistance
5. Push the project to GitHub

## Goal

The goal is to create a curved pavilion surface and place repeated shingle panels on it using parameters.

## Parameters

- Pavilion span
- Pavilion height
- Pavilion length
- Row count
- Column count
- Shingle width
- Shingle height
- Row offset
- Support height

## Logic

The script creates a curved pavilion using points and a simple mathematical curve. Then it creates a grid of points across the pavilion. Every second row is offset to create a staggered shingle pattern. Finally, the script counts and stores all generated shingles.
