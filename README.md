# Parametric Curved Shingle Pavilion

**Student Name:** Baran Koç

## Project Description

This project is a parametric curved shingle pavilion generated with Grasshopper and Python.

The design is based on a top-down computational workflow. First, the global pavilion surface is defined. Then the surface is divided into a grid. Each grid cell receives a repeated shingle panel. The shingles are staggered, overlapped, tilted outward, and supported by a structural frame.

## Goal

The goal is to create a parametric pavilion system where the overall form, shingle density, shingle size, overlap, tilt angle, and support structure can be controlled through Grasshopper sliders.
