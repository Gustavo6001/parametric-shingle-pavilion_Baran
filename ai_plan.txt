AI DEVELOPMENT PLAN

PROJECT:
Parametric Curved Shingle Pavilion

OBJECTIVE:
Use AI assistance to develop a Grasshopper Python script that generates a curved pavilion covered with repeated shingle panels.

PLAN:

1. Define the design goal.
   The project will generate a curved vertical pavilion surface covered with repeated shingles.

2. Write detailed pseudo-code.
   The pseudo-code explains the computational logic before writing the actual code.

3. Identify the main input parameters.
   The inputs include pavilion size, curvature, grid density, shingle size, overlap, tilt angle, support height, and base dimensions.

4. Create the global pavilion surface.
   The pavilion surface is generated using a mathematical function. Width controls the horizontal direction, height controls the vertical direction, and depth plus curvature create the bending effect.

5. Divide the surface into a grid.
   The grid creates placement positions for the shingles.

6. Create local frames.
   At every grid point, the script calculates local directions and a surface normal. These are used to orient the shingles.

7. Generate the fish-scale pattern.
   Every second row is shifted horizontally to create a staggered shingle pattern.

8. Create the shingle geometry.
   Each shingle is created as a panel and placed on the local surface frame.

9. Add tilt and depth.
   The shingles are tilted outward to create shadow and depth, similar to architectural shingle facade references.

10. Add supports and structure.
    The project includes support spacers, vertical ribs, horizontal purlins, side frames, and a base platform.

11. Test the script in Grasshopper.
    Sliders are connected to the Python inputs. The outputs are previewed in Rhino.

12. Organize the project files.
    The repository includes README.md, pseudo_code.txt, ai_plan.txt, main.py, and requirements.txt.

13. Use GitHub Desktop.
    The project folder is committed and pushed to GitHub.

14. Share the repository.
    The final repository link is submitted as proof of the computational workflow.
